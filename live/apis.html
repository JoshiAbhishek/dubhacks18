    • PortfolioAnalysis

    Portfolio analysis provides the ability to run analysis on a bespoke portfolio. You can get financial data like risk, performance, exposures, etc. on a group of financial assets like stocks, ETFs, mutual funds, etc.


    PerformanceData

    Performance data captures the loss and gains of securities over a given period. Key performance data points include rates of returns, which is your profit on your investment over a specified period of time.


    SearchSecurities

    Security search allows you to search the universe of globally available securities by description (i.e. "blackrock stock"), ticker (i.e. "BLK"), or any other field - or combination of fields - available in the data.


    SecurityData

    Security data retrieves analytics, breakdowns, characteristics, identifiers, and some performance metrics for a collection of securities. The only required parameter is a list of identifiers (preferably prefixed by identifier type), i.e. ticker:IVV, ticker:AAPL, ticker:VWO
